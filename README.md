## 🇫🇷 Français

### 🌟 À propos de ce projet
Ce dépôt contient mon portfolio professionnel, conçu pour mettre en avant une double expertise rare : le développement de solutions administratives sécurisées (**FinTech**) et la surveillance réseau opérationnelle (**Cybersécurité**).

### 🛠️ Projets mis en avant
- **Gestion des Œuvres Sociales** : Une plateforme complète (Python/Flask) incluant un Assistant IA, un journal d'audit (Audit Trail) et un générateur de fichiers bancaires VIRM.
- **Zana Cyber-Agent** : Monitoring en temps réel via **Security Onion** pour l'analyse de paquets et la détection d'intrusions.
- **Coffre-fort Numérique** : Utilisation d'**OpenSSL** et de l'algorithme RSA pour le chiffrement asymétrique de données sensibles.
- **Analyse de Données** : Scripts de traitement de données massives avec **Pandas, NumPy et Scikit-Learn**.

### 💻 Technologies utilisées
- **Frontend** : HTML5, Tailwind CSS (Design moderne et responsive).
- **Backend** : Python, Flask, SQLite.
- **Data** : Matplotlib, Seaborn (Visualisation).
- **Sécurité** : OpenSSL, Security Onion, IDS.

### 🌐 Déploiement
Le portfolio est accessible en ligne via GitHub Pages à l'adresse suivante : `https://bekheddazana.github.io`
